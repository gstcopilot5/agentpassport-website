⚡ Integration Guide

LangChain + AgentAudit

Add trust, logging, and real-time safety checks to your LangChain agents in under 5 minutes. Every tool call gets authorized, logged, and analyzed automatically.

Installation

Install the AgentAudit SDK alongside your LangChain packages.

bash
npm install @agentaudit/sdk langchain @langchain/openai

Basic Setup

Initialize AgentAudit and register your agent before making any LangChain calls.

1
Initialize the SDK
javascript
const AgentAudit = require('@agentaudit/sdk');

const audit = new AgentAudit(process.env.AGENTAUDIT_API_KEY);
2
Register your LangChain agent
javascript
const agent = await audit.agents.register({
  name: 'customer-support-agent',
  description: 'Handles customer queries via LangChain'
});

// Store this ID — you'll use it for every execution
const agentId = agent.id;
3
Wrap tool calls with execute()
javascript
const { ChatOpenAI } = require('@langchain/openai');
const { DynamicTool } = require('langchain/tools');

// Wrap any LangChain tool with AgentAudit
function auditedTool(name, description, handler) {
  return new DynamicTool({
    name,
    description,
    func: async (input) => {
      // Log + authorize through AgentAudit
      const result = await audit.execute.run({
        agentId: agentId,
        action: name,
        params: { input }
      });

      // If blocked by policy, return reason
      if (!result.allowed) {
        return `Action blocked: ${result.reason}`;
      }

      // Otherwise run the actual tool
      return await handler(input);
    }
  });
}

Adding Safety Policies

Define rules that automatically block dangerous actions before your agent executes them.

javascript
// Block emails to external domains
                                                                                                                                                                                                                                                                                                                                                                                          await audit.policies.create({
                                                                                                                                                                                                                                                                                                                                                                                            agentId,
                                                                                                                                                                                                                                                                                                                                                                                              name: 'no-external-emails',
                                                                                                                                                                                                                                                                                                                                                                                                rule: 'block',
                                                                                                                                                                                                                                                                                                                                                                                                  action: 'send_email',
                                                                                                                                                                                                                                                                                                                                                                                                    condition: 'params.to not contains @yourcompany.com'
                                                                                                                                                                                                                                                                                                                                                                                                    });
                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                    // Rate limit web searches to 10/minute
                                                                                                                                                                                                                                                                                                                                                                                                    await audit.policies.create({
                                                                                                                                                                                                                                                                                                                                                                                                      agentId,
                                                                                                                                                                                                                                                                                                                                                                                                        name: 'search-rate-limit',
                                                                                                                                                                                                                                                                                                                                                                                                          rule: 'rate_limit',
                                                                                                                                                                                                                                                                                                                                                                                                            action: 'web_search',
                                                                                                                                                                                                                                                                                                                                                                                                              limit: 10,
                                                                                                                                                                                                                                                                                                                                                                                                                window: '1m'
                                                                                                                                                                                                                                                                                                                                                                                                                });
⚠
Policies are evaluated before every execute.run() call. Blocked actions return allowed: false with a reason — your agent gracefully handles this instead of crashing.

SDK Reference

All methods available in @agentaudit/sdk.

Method Description
audit.agents.register(data) Register a new agent, returns { id, name }
audit.execute.run(data) Run action through 7 security checks
audit.policies.create(data) Create a policy rule for an agent
audit.audit.forAgent(id) Get full execution history for agent
audit.audit.summary(id) AI-generated behavioral summary (Gemini)
audit.risk.profile(id) Get risk score and flags for agent
audit.incidents.list() List all detected incidents
